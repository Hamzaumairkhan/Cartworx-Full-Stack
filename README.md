# 🛒 CartWorx – MERN Stack E-Commerce Platform

A modern, full-stack e-commerce web application built using the **MERN Stack**. CartWorx provides a seamless online shopping experience with secure authentication, product management, cart functionality, order processing, and an intuitive admin dashboard.

## 🌐 Live Demo

**Frontend:** https://cartworx-frontend.vercel.app/

---

# ✨ Features

### 👤 User Features

* 🔐 Secure User Authentication (JWT)
* 📝 User Registration & Login
* 🛍️ Browse Products
* 🔎 Search & Filter Products
* 📄 Product Details Page
* ❤️ Wishlist Support
* 🛒 Shopping Cart
* ➕ Add / Remove Cart Items
* 📦 Place Orders
* 📜 Order History
* 👤 User Profile Management
* 📱 Fully Responsive Design

---

### 🛠️ Admin Features

* Admin Dashboard
* Add Products
* Edit Products
* Delete Products
* Manage Categories
* Manage Orders
* Manage Users
* Inventory Management
* Sales Overview

# 🚀 Tech Stack

## Frontend

* React.js
* React Router DOM
* Redux Toolkit / Context API
* Axios
* Tailwind CSS
* Vite

## Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt.js
* Multer
* Cloudinary (if used)

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/cartworx.git
```

---

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=Your MongoDB Connection String

JWT_SECRET=Your Secret Key

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
